Outgoing Email Server Configuration:

-> Run locahost:8069
-> Activate Debugging mode

-> Navigate: Settings/Technical/Email/Outgoing Mail Server,

	     Create New: 
			 Authenticate with: Username
			 Check Mark TLS (STARTTLS)
			 SMTP Server: smtp.gmail.com
			 Port: 587
			 Username: <your email >
			 Password: <use app specific password, refer the link given at the end>
			 Check Mark Debugging
	     
		Then click 'Test Connection'
